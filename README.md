Simple app to practice uploading files in React and GO

# Running the Application

## Frontend

- In a terminal:
    - cd frontend/fileupload
    - npm install
    - npm start
    - Access the UI at http://localhost:3000/app

## Backend

- In a terminal:
    - cd backend
    - go run main.go
